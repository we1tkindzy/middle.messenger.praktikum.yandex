## Описание
- авторизоваться
- отправлять сообщения
- редактировать профиль
- добавлять и удалять контакты

## Установка
- `npm i` — установка стабильной версии
- `npm run start` — запуск версии для разработчика
- `npm run build` — сборка на 3000 порту
